# 🚀 Open Source

## Blender

I actively contribute to **Blender's** C++ codebase, working on rendering and editor systems.

### ✅ Merged Contribution

**Fix #160789 — Crash when moving Grease Pencil material slot with invalid active material index**

While investigating a reported crash, I traced the issue to an invalid active material index that propagated through Blender's material management system.

The final solution involved:

- Debugging Blender's C++ internals
- Finding the true root cause instead of only fixing the symptom
- Refactoring Grease Pencil-specific logic into a reusable kernel utility
- Improving material index sanitization during blend file loading
- Collaborating with Blender maintainers through multiple review iterations

The contribution was merged into **Blender 5.2**.

# 🛠 Featured Project

## BerryFlux

> A modern open-source C++ Game Engine built completely from scratch.

BerryFlux is my long-term project focused on learning and experimenting with graphics programming, rendering, engine architecture, and tooling.

### Current Features

- Event System
- Layer Architecture
- Window Abstraction
- OpenGL Renderer
- Renderer2D
- Shader Library
- Material System
- Texture System
- Orthographic Camera
- ImGui Dockspace
